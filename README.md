# summarize-gpt
Summarrize your articles with Open AI's GPT-4 !
**************
To run locally :
*************
Obtain API keys from the link below
*************
API KEY = https://rapidapi.com/restyler/api/article-extractor-and-summarizer
*****************
In your .env file add:
VITE_Rapid_APIKey ="API KEY"
*****************
npm install
*******
npm run dev
